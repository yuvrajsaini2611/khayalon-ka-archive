# Khayalon Ka Archive

Khayalon Ka Archive is a personal blog built to share thoughts, reflections, poetry, and meaningful ideas in a clean and engaging reading experience.

## Live Demo

https://khayalon-ka-archive.vercel.app/

## Features

* Responsive design for desktop and mobile devices
* Blog post navigation using React Router
* Clean and minimal reading experience
* Custom branding with logo and favicon
* Fast performance powered by Vite

## Built With

* React
* Vite
* React Router
* Tailwind CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/yuvrajsaini2611/khayalon-ka-archive.git
```

Navigate to the frontend directory:

```bash
cd khayalon-ka-archive/frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
